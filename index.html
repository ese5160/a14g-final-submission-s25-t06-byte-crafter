2. Summary

This system uses a fingerprint sensor with normal and duress modes, integrates IMU, servo motor, buzzer, LCD, rotary encoder, and Wi-Fi communication via SAMW25 MCU.

3. Requirements

  • ✔️ Fingerprint unlock with cloud-controlled add/delete
  • ✔️ IMU tamper detection with buzzer alert
  • ✔️ Servo motor for lock actuation
  • ✔️ Rotary encoder and LCD for user interface
